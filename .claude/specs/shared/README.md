# Shared Specifications

共有（クロスカッティング）ドキュメント。複数の機能に渡る設計・仕様を定義します。

## Documents

| Document | Description |
|----------|-------------|
| [architecture.md](./architecture.md) | システムアーキテクチャ |
| [data-models.md](./data-models.md) | データモデル定義 |
| [security.md](./security.md) | セキュリティ設計 |
| [testing.md](./testing.md) | テスト戦略 |

## Purpose

これらのドキュメントは特定の機能に限定されず、システム全体に適用される設計方針を定義します。

- **Architecture**: プラットフォーム検出、MCP Server構造、依存関係
- **Data Models**: 共通データ型、インターフェース定義
- **Security**: 認証・認可の全体設計、脅威モデル
- **Testing**: テスト方針、カバレッジ目標、CI/CD設定
